What is WURFL?

WURFL stands for Wireless Universal Resource File. It is an open source project that contains information on every wireless device on earth. WURFL.js is a JavaScript that detects device models of smartphones, tablets, smart TVs, and game consoles accessing

Capability Description
advertised_browser This virtual capability infers the name of the browser based on user-agent string analysis (and possibly the analysis of other HTTP headers and WURFL capabilities)
device_os A static capability that identifies the device’s host operating system.
form_factor A virtual capability that returns the device’s form factor based on categorization provided by multiple capability tests.
is_app A virtual capability that returns true if the requesting HTTP client is an app.
is_robot A virtual capability that identifies whether the device is a robot or not.
